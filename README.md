
# News App

This news app is a mordern app which fetch the data from newsApi.org and render it.


I have used api of https://newsapi.org/  website. Please create account on that website and get the api key.
## Installation and Running
Before Installation get the api key store it in an .env file, add this .env file along with public and src folder.

.env content ->
REACT_APP_NEWS_API_KEY='your API key'

To run the application, run the following command

```bash
  npm i 
  npm start
```

npm i - to install all the libraries used

npm start - to run the application
